# Student-Management-System

📌 Overview
A Java‑based Student Management System designed to handle student records efficiently. 
This project demonstrates object‑oriented design, Oracle database integration via JDBC, 
and encapsulation with validation for professional, recruiter‑ready code.

✨ Features
- Add, update, delete, and view student records
- Encapsulation with getters/setters and validation
- Custom exceptions for error handling
- JDBC integration with Oracle database
- SQL queries replacing ArrayList logic for persistence

 🛠️ Tech Stack
- Language: Java
- Database: Oracle SQL
- Tools: JDBC, VS Code
- Concepts: OOP, encapsulation, exception handling
  
⚙️ How to Run
- Clone the repository:
        git clone https://github.com/your-username/StudentManagementSystem.git
- Configure Oracle database and update JDBC connection string in DBConnection.java.
- Compile and run:
    javac *.java
    java Main
